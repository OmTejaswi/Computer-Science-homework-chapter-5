Defination of some common terms related to computer

Data
Data means raw facts and figures.
Information
Processed data in a meaningful form is called Information
Internet, World Wide Web
The World Wide Web is a system that operates via this wiring. Web pages are transmitted via Internet connections but there is more to the Internet than just the web. Many other types of data travel across the Internet too, for example email.
Web Browser
A program you use to look at, and navigate between, pages on the World Wide Web. Examples include Internet Explorer and Firefox although there are many others. Again, people sometimes refer to web browsers as “the Internet”, whereas they really only provide the means to view pages on the web.
